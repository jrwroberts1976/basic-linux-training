# Linux Training Plan

## Overview

This training plan provides a structured introduction to Linux system administration.

The aim is to give engineers the practical skills required to work confidently with Linux servers, troubleshoot issues, and support production environments.

---

## Target Audience

This training is aimed at:

- New Linux users
- Junior system administrators
- Support engineers
- Engineers moving from Windows-based environments
- Cloud engineers requiring Linux skills

---

## Training Objectives

By completion of this training, participants should be able to:

- Navigate and manage Linux systems from the command line
- Understand Linux filesystem structure
- Manage users, groups and permissions
- Monitor processes and services
- Install and manage software packages
- Configure basic networking
- Understand storage concepts
- Write basic Bash scripts
- Troubleshoot common Linux issues
- Apply basic Linux security practices

---

3. Check logs
