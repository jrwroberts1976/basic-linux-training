# Overview

Understanding how Linux organises files and controls access is a fundamental skill for every Linux administrator.

In this module you will learn how the Linux filesystem is structured, how file ownership and permissions work, and how to locate files efficiently. These skills are essential for managing systems securely and troubleshooting everyday administration tasks.

---

# Learning Objectives

By the end of this module you will be able to:

- Describe the Linux filesystem hierarchy.
- Identify the purpose of common system directories.
- Understand file ownership and group ownership.
- Read and interpret Linux file permissions.
- Modify permissions using `chmod`.
- Change ownership using `chown`.
- Locate files using common search tools.
- Create and use symbolic links.
