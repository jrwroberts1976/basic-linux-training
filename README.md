# Basic Linux Training

A structured Linux training course designed to teach the fundamentals of Linux system administration through hands-on lessons, practical exercises and real-world administration tasks.

---

# Purpose

This repository provides a structured learning path covering:

- Linux fundamentals
- Command-line administration
- Filesystem management
- User and group administration
- Process management
- Package management
- Networking
- Storage
- Bash scripting
- Linux security
- Troubleshooting

The aim is to provide practical Linux skills for engineers working in modern IT environments.

---

# Audience

This training is aimed at:

- New Linux users
- Junior system administrators
- Service Desk and Support Engineers
- Engineers transitioning from Windows to Linux
- Cloud and Infrastructure Engineers
- Anyone wishing to learn Linux administration

No previous Linux administration experience is required.

# Recommended Training Environment

The course can be completed using any Linux installation.

Recommended environments:

- Virtual Machine (VirtualBox, VMware or Hyper-V)
- Raspberry Pi
- Cloud Virtual Machine
- Windows Subsystem for Linux (WSL)

Recommended distributions:

- Ubuntu Server LTS
- Debian
- Rocky Linux
