# Basic Linux Training

A structured Linux training repository for team members learning Linux administration.

## Purpose

This repository provides:

- Linux fundamentals
- Command-line training
- System administration concepts
- Practical exercises
- Troubleshooting guidance

## Audience

Designed for:

- New Linux users
- Junior administrators
- Support engineers
- Engineers transitioning to Linux environments

## Training Modules

| Module | Topic |
|---|---|
| 00 | Introduction to Linux |
| 01 | Command Line Basics |
| 02 | Filesystem |
| 03 | Users and Groups |
| 04 | Processes and Services |
| 05 | Package Management |
| 06 | Networking |
| 07 | Storage |
| 08 | Bash Scripting |
| 09 | Security |
| 10 | Troubleshooting |
