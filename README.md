# Basic Linux Training

A structured Linux training repository for team members learning Linux administration.

---

# Purpose

This repository provides a structured learning path covering:

- Linux fundamentals
- Command-line administration
- System management
- Networking
- Storage
- Security
- Troubleshooting
- Automation

The aim is to provide practical Linux skills for engineers working with modern IT environments.

---

# Audience

This training is aimed at:

- New Linux users
- Junior system administrators
- Support engineers
- Engineers transitioning to Linux environments
- Cloud and infrastructure engineers

No previous Linux administration experience is required.

# Recommended Training Environment

Training can be completed using:

- Virtual machines
- Raspberry Pi
- Cloud instances
- Windows Subsystem for Linux (WSL)

Recommended Linux distributions:

- Ubuntu Server LTS
- Debian
- Rocky Linux

# Learning Approach

Each module contains:

- 📖 Theory
- 💻 Command examples
- 🧪 Practical exercises
- ✅ Knowledge checks

The aim is to learn Linux by understanding concepts and applying them.
