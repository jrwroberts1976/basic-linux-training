# Module 00 - Introduction to Linux

## Overview

Welcome to the **Basic Linux Training** course.

This module introduces the Linux operating system and provides the foundation knowledge required before starting Linux administration.

You will learn about:

- What Linux is
- How Linux works
- Linux history
- Linux distributions
- Linux server environments
- How to access Linux systems

---

# Learning Objectives

By completing this module, you will understand:

- What Linux is and why it is widely used
- The role of the Linux kernel
- How Linux differs from other operating systems
- The history behind Linux
- What Linux distributions are
- The difference between desktop and server Linux
- How to access a Linux system
- Basic Linux terminology
