# Linux History

## UNIX

Linux was inspired by UNIX, an operating system created at Bell Labs in the 1970s.

UNIX introduced concepts still used today:

- Multi-user systems
- File permissions
- Command-line administration
- Small tools working together

---

## GNU Project

The GNU Project started in 1983 with the goal of creating a free UNIX-like operating system.

GNU provided many important tools used by Linux today:

- Bash shell
- Core utilities
- Compilers
- Libraries

---

## Linux Kernel

In 1991, Linus Torvalds released the first Linux kernel.

The Linux kernel combined with GNU tools created what most people refer to as a Linux operating system.

Linux grew because:

- Source code was freely available
- Developers worldwide contributed
- Organisations adopted it

Today Linux powers everything from embedded devices to cloud platforms.
